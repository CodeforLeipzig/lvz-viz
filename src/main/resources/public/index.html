Häufigkeit und Orte der im Polizeiticker (LVZ) beschriebenen Fälle in Leipzig.
Verändern Sie den Slider um verschiedene Zeitfenster zu sehen.
Die hier abgebildeten Daten entstammen prototypischen Algorithmen, welche Orte aus Texten extrahieren. Das Mapping der Orte auf die Karte erfolgt über die Novatim API von Open Street Map. Vorhandene Fehler bitten wir zu entschuldigen.
Auf der Karte werden die Orte dargestellt, welche aus den LVZ Polizeiticker Artikeln erkannt wurden.
Sie können auch über die Suche nach speziellen Orten und Begriffen suchen.
